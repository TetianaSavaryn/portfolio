Saint Sophie website was created for the St. Sophie parish located at St. Sophie Cathedral in Montreal. The project had to represent different organizations as parts of the structure of the parish. This required big photo galleries with hovers and previews.

The website represents a ukrainian community in Quebec, Canada. So it was necessary to include a functional translation between english, ukrainian and french languages. The internationalisation was done with jQuery I18n.

Some other features included in the website: a library of PDF bulletins, calendar, Google forms, audio player.
