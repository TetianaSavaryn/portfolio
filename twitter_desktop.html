Twitter Desktop is a complete desktop application with user registration and authentification, customized sign-in and log-in UI. Each user is allowed to post, edit and delete own tweets. He can read but not edit or delete all the other users' tweets. The app also provides the 'contact us' feature for registered users, tweets counter and data stamps.

Login and Signup were implemented with JavaFX and JDBC. The project was built with MVC architecture using Material Design Library.

Intuitive UI was presented with some JavaFX Effects and Transformations features.
